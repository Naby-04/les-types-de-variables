Les boucles en javascript

Les boucles en JavaScript sont des structures qui permettent d'exécuter plusieurs fois une série d'instructions. Elles sont très utiles pour traiter des ensembles de données, parcourir des tableaux ou des objets, et automatiser des tâches répétitives. Voici la liste des boucles en JavaScript, avec leur syntaxe, leurs utilités, et les situations où il est recommandé de les utiliser.

La boucle for est idéale pour les itérations quand on connaît le nombre exact de répétitions à effectuer, comme dans le cas de tableaux ou de listes. Elle est souvent utilisée pour parcourir des tableaux ou exécuter des instructions un nombre fixe de fois.

syntaxe: 
                        
for (initialisation; condition; incrément) {
  // Code à exécuter
}
Exemple: 
                        
for (let i = 0; i < 5; i++) {
  console.log("Itération numéro :", i);
}

while est une boucle qui s'exécute tant qu'une condition spécifique est vraie. Elle est idéale quand on ne connaît pas exactement le nombre d'itérations à l'avance, mais qu'on a une condition à respecter pour arrêter la boucle.

syntaxe: 
                        
while (condition) {
  // Code à exécuter
}
Exemple: 
                        
let compteur = 0;
while (compteur < 5) {
  console.log("Compteur :", compteur);
  compteur++;
}

La boucle do...while est similaire à la boucle while, mais elle garantit que le bloc de code sera exécuté au moins une fois avant de vérifier la condition. Elle est utile si on veut exécuter des instructions au moins une fois, même si la condition est fausse dès le départ.

syntaxe: 
                        
do {
  // Code à exécuter
} while (condition);
Exemple: 
                        
let compteur = 0;
do {
  console.log("Compteur :", compteur);
  compteur++;
} while (compteur < 5);

La boucle for...of est principalement utilisée pour parcourir les éléments d'un tableau ou d'une collection itérable. Elle est utile quand on veut accéder directement aux valeurs des éléments d'un tableau sans utiliser d'index.

syntaxe: 
                        
for (variable of iterable) {
  // Code à exécuter
}
Exemple: 
                        
let fruits = ["pomme", "banane", "poire"];
for (let fruit of fruits) {
  console.log(fruit);
}

La boucle for...in est utilisée pour parcourir les propriétés d'un objet. Elle est utile pour accéder directement aux clés d'un objet, ce qui est pratique pour manipuler des objets dont les propriétés ne sont pas nécessairement connues à l'avance.

syntaxe: 
                        
for (clé in objet) {
  // Code à exécuter
}
Exemple: 
                        
let personne = { nom: "Jean", age: 30, ville: "Paris" };
for (let propriété in personne) {
  console.log(propriété, ":", personne[propriété]);
}
